Risk Assessment Model

Python script for calculating Value at Risk (VaR).


                    import numpy as np
                    
                    def calculate_var(returns, confidence_level=0.95):
                        # Calculate VaR at a given confidence level
                        return np.percentile(returns, 100 * (1 - confidence_level))

                    # Example usage
                    portfolio_returns = np.random.normal(-0.001, 0.01, 1000)
                    var_95 = calculate_var(portfolio_returns)
                    print(f"95% VaR: {var_95:.4f}")

We utilize the Black-Scholes formula for European options:

$$ C(S, t) = N(d_1)S - N(d_2)Ke^{-r(T-t)} $$

Where:

  • $C$ = Call option price
  • $S$ = Current stock price
  • $K$ = Option strike price
Our derivatives pricing engine is built on standard models like Black-Scholes. This formula is fundamental to how we price call options and manage risk.
